Sobre o Alugarzin

O Alugarzin é um sistema Web completo para divulgação, busca e gerenciamento de imóveis para aluguel ou venda.

O projeto inclui autenticação JWT, cadastro e login de usuários, publicação de anúncios, visualização detalhada, edição e exclusão, além de funcionalidades de contato entre interessados.
